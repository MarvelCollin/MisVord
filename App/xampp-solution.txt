QUICK SOLUTION - USE XAMPP INSTEAD OF IIS:

1. Download XAMPP: https://www.apachefriends.org/download.html

2. Install XAMPP

3. Copy project ke: C:\xampp\htdocs\misvord\

4. Start Apache dan MySQL dari XAMPP Control Panel

5. Access: http://localhost/misvord/test-iis.php

PROS:
- PHP sudah terinstall
- MySQL included
- Easier setup
- Good for development

CONS:
- Not using IIS
- Apache instead

FOR PRODUCTION IIS:
- Follow install-php-iis.txt
- Or use Windows Server with PHP pre-installed
