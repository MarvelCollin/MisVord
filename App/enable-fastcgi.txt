ENABLE FASTCGI IN WINDOWS:

METHOD 1 - Via Windows Features:
1. Open "Turn Windows features on or off"
2. Expand "Internet Information Services"
3. Expand "World Wide Web Services"  
4. Expand "Application Development Features"
5. Check ✓ "CGI" 
6. Check ✓ "FastCGI" (if available)
7. Click OK and restart

METHOD 2 - Via Command Line (Run as Administrator):
dism /online /enable-feature /featurename:IIS-CGI
dism /online /enable-feature /featurename:IIS-ASPNET45

METHOD 3 - Via PowerShell (Run as Administrator):
Enable-WindowsOptionalFeature -Online -FeatureName IIS-CGI
Enable-WindowsOptionalFeature -Online -FeatureName IIS-ASPNET45

After enabling, restart computer and check IIS Manager again.
