INSTALL PHP FOR IIS STEP BY STEP:

1. Download PHP for Windows:
   https://windows.php.net/download/
   - Pilih "Non Thread Safe" untuk IIS
   - Download versi 8.1 atau 8.2

2. Extract ke folder: C:\php

3. Download Web Platform Installer:
   https://www.microsoft.com/web/downloads/platform.aspx

4. Install via Web Platform Installer:
   - Cari "PHP"
   - Install "PHP 8.1.x (x64)" untuk IIS

5. Atau install manual:
   - Copy php.ini-production ke php.ini
   - Edit php.ini:
     extension_dir = "ext"
     extension=mysqli
     extension=pdo_mysql

6. Add PHP ke IIS:
   - Open IIS Manager
   - Click "Handler Mappings"
   - Add Module Mapping:
     Request path: *.php
     Module: FastCgiModule
     Executable: C:\php\php-cgi.exe
     Name: PHP

7. Restart IIS: iisreset

ALTERNATIVE - Use XAMPP:
1. Download XAMPP from apachefriends.org
2. Install (includes PHP + MySQL)
3. Use Apache instead of IIS
