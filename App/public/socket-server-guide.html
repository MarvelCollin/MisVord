üîß Troubleshooting

If connection fails:
  1. Make sure the socket server is running (Step 1 above)
  2. Check that port 1002 is not blocked by firewall
  3. Verify no other service is using port 1002
  4. Try restarting the socket server
# Check if port 1002 is in use:
netstat -ano | findstr :1002
